#  初识JavaWbe

## 什么是Javaweb

### web

web称全球广域网，也称万维网（www），能够通过浏览器访问的网站

### javaWeb  
用java技术解决相关web互联网领域的技术栈（开发互联网的应用程序）

![](./img/01-1.png)
*JavaWeb基本了解*
- 网页：负责展现数据
- 数据库：存储和管理数据
- JavaWeb程序：逻辑处理
